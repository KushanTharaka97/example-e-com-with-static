## 📖 Overview

**Gold Lanka** is a modern, responsive e-commerce platform showcasing exquisite 22K and 24K gold jewelry. Built with a focus on performance and user experience, this static website demonstrates a premium digital storefront for high-end luxury goods.

The project highlights traditional craftsmanship through a clean, elegant interface that allows users to browse collections, view detailed product specifications, and manage a wishlist.

## 🚀 Features

- **💎 Premium UI/UX**: minimalist design with focus on product imagery and typography.
- **📱 Fully Responsive**: Optimized for mobile, tablet, and desktop devices.
- **🔎 Dynamic Catalog**: Filter products by Category (Necklaces, Rings, etc.) and Gold Karat (22K, 24K).
- **⚡ Fast Performance**: Pure HTML/CSS/JS with no heavy framework overhead.
- **❤️ Wishlist Functionality**: LocalStorage-based wishlist to save favorite items.
- **🌍 Localization Ready**: Structure in place for multi-language support (EN, SI, TA).
- **♿ Accessible**: Built with semantic HTML5 and ARIA roles for better accessibility.

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3 (Custom Properties, Flexbox, Grid), Vanilla JavaScript (ES6+).
- **Typography**: Google Fonts (Cormorant Garamond for elegance, Inter for readability).
- **Icons**: SVG Icons & Unicode symbols.
- **Data**: JSON-based product catalog.

## 🏁 Getting Started

To run this project locally, you don't need a complex build step.

1.  **Clone the repository**
    ```bash
    git clone https://github.com/your-username/gold-shop.git
    cd gold-shop
    ```

2.  **Serve the application**
    *   You can use any static file server.
    *   **VS Code**: Right-click `index.html` and select "Open with Live Server".
    *   **Python**: `python -m http.server 8000`
    *   **Node**: `npx serve .`

3.  **Open in Browser**
    *   Navigate to `http://localhost:8000`
